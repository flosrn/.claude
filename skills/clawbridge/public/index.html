Configure Permanent Link

To keep your URL fixed, you need a Cloudflare Tunnel Token.

Where to find your Token?
1. Go to Cloudflare Dash > Networking > Tunnels.
2. Create a tunnel, choose "Docker" (easiest to copy) or your OS, and grab the token string (starts with ey...).
3. After install, go to Public Hostname in your Tunnel settings and map your domain to localhost:3000.
Option 1: Ask your Agent (Chat)
Send this message to OpenClaw:
Run this command to fix my dashboard URL:
cd skills/clawbridge && ./install.sh --token=YOUR_TOKEN
📋
Option 2: Terminal Command
cd ~/skills/clawbridge && ./install.sh --token=YOUR_TOKEN
